# **App Name**: StitchStyle

## Core Features:

- Product Browsing: Browse products by category, filter by attributes, and sort by price or popularity.
- Custom Order Submission: Submit custom clothing requests with design images, measurements, and fabric preferences.
- Secure Checkout: Checkout with multiple payment options and address management.
- Order Tracking: Track order status and delivery progress.
- Admin Product Management: Add, edit, and delete products, manage stock levels, and upload multiple images.
- Admin Order Management: View, process, and update orders with status updates and payment confirmations.
- Style Suggestion Tool: AI tool provides personalized style recommendations based on customer preferences and trending fashion in Nigeria.

## Style Guidelines:

- Primary color: Muted emerald (#50C878) to reflect the brand's elegance.
- Background color: Ivory (#FFFFF0), offering a warm, clean feel that helps highlight the product imagery.
- Accent color: Soft gold (#D4AF37), drawing attention to key elements like buttons and calls to action, further reflecting a premium brand aesthetic.
- Body and headline font: 'Montserrat', a geometric sans-serif for readability and class.
- Minimal and elegant icons to enhance user navigation and experience.
- Grid-based layout for product listings and card-based design for featured items.
- Subtle animations for transitions and user interactions, creating a dynamic user experience.
